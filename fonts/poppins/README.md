# Polices Poppins pour JNV Marketing

## Fichiers nécessaires

Vous devez télécharger les fichiers suivants au format WOFF2 :

1. **Poppins-Regular.woff2** - Pour le texte normal (font-weight: 400)
2. **Poppins-Medium.woff2** - Pour le texte medium (font-weight: 500)
3. **Poppins-SemiBold.woff2** - Pour le texte semi-gras (font-weight: 600)
4. **Poppins-Bold.woff2** - Pour le texte gras (font-weight: 700)
5. **Poppins-Light.woff2** - Pour le texte léger (font-weight: 300, optionnel)

## Comment télécharger ces fichiers

### Option 1 : Télécharger depuis Google Fonts
1. Aller sur [Google Fonts Poppins](https://fonts.google.com/specimen/Poppins)
2. Sélectionner les variantes de poids souhaitées (300, 400, 500, 600, 700)
3. Télécharger les fichiers WOFF2 pour chaque variante

### Option 2 : Utiliser un service de conversion
1. Aller sur [google-webfonts-helper](https://gwfh.mranftl.com/fonts/poppins)
2. Sélectionner les variantes de poids souhaitées
3. Choisir le format WOFF2
4. Télécharger les fichiers

### Option 3 : Télécharger directement depuis Google
Les fichiers peuvent être téléchargés directement depuis les URLs suivantes :
- Regular: https://fonts.gstatic.com/s/poppins/v20/pxiEyp8kv8JHgFVrJJfecg.woff2
- Medium: https://fonts.gstatic.com/s/poppins/v20/pxiByp8kv8JHgFVrLGT9Z1xlFQ.woff2
- SemiBold: https://fonts.gstatic.com/s/poppins/v20/pxiByp8kv8JHgFVrLEj6Z1xlFQ.woff2
- Bold: https://fonts.gstatic.com/s/poppins/v20/pxiByp8kv8JHgFVrLCz7Z1xlFQ.woff2
- Light: https://fonts.gstatic.com/s/poppins/v20/pxiByp8kv8JHgFVrLDz8Z1xlFQ.woff2

## Comment utiliser les polices

Placez les fichiers .woff2 dans ce dossier. Le système les utilisera automatiquement grâce au fichier fonts.css qui a déjà été configuré.

## Avantages de cette approche
- Chargement plus rapide du site
- Pas de requêtes externes à Google Fonts
- Meilleur score de performance SEO
- Conforme RGPD (pas de transfert de données vers Google)
